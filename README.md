# 🚀 Dashboard Builder

A modern, professional dashboard construction kit built with **Vanilla JavaScript**, **Firebase**, and **Chart.js**. Create, customize, and persist data-driven dashboards with ease.

---

## ✨ Key Features

- **🏗️ Live Workspace Customization**: 12-column grid system for drag-and-drop widget placement.
- **📊 Intelligent Data Visualizations**: Multi-type charts (Area, Scatter, Pie) powered by Chart.js.
- **☁️ Real-time Cloud Sync**: Instant persistence across sessions using Firebase Firestore.
- **🎛️ Granular Widget Control**: Editable axis labels, data metrics (Sum/Avg/Count), and UI styles.
- **🔐 Secure Environment**: Full authentication flow with Firebase Auth and protected views.
- **📅 Temporal Analytics**: Quick-switch filters for "Today," "Last 7 Days," or "All Time."

## 🛠️ Tech Stack

- **Frontend**: Vanilla JavaScript (ES6+), HTML5, CSS3
- **Database**: Firebase Firestore (Real-time storage)
- **Authentication**: Firebase Auth (Secure login system)
- **Charts**: Chart.js (Interactive data visualization)
- **Styling**: Custom CSS (Flexbox, CSS Grid, CSS Variables)
- **Icons & Fonts**: SVG icons, Inter Font (Google Fonts)

---

## 📂 Project Structure

- `index.html`: Main application entry point and structural layout.
- `script.js`: Core logic for drag-and-drop, data fetching, and chart rendering.
- `style.css`: Modern, responsive UI styling using a comprehensive design system.

## 🚀 Getting Started

1. **Clone the repository**:
   ```bash
   git clone https://github.com/Vasanthvelumani/Vasanth-Dashboard-builder.git
   ```
2. **Setup Firebase**:
   - Create a Firebase project at [Firebase Console](https://console.firebase.google.com/).
   - Enable Firestore and Authentication (Email/Password).
   - Add your Firebase configuration keys to the appropriate section in `script.js`.
3. **Open the project**:
   - Simply open `index.html` in your favorite browser or use a local development server (e.g., Live Server).
