# Dashboard Builder

![Version](https://img.shields.io/badge/version-1.0.0-blue)

Dashboard Builder is a dynamic, full-stack web application designed to transform raw data into actionable insights through a highly customizable interface. Built with a focus on user autonomy, it allows users to construct personalized workspaces using an intuitive drag-and-drop engine.

The core of the application lies in its ability to bridge the gap between data entry and high-level visualization. Users can manage the entire lifecycle of customer orders—from creation and editing to real-time tracking—while simultaneously building a visual layer of KPIs, Bar Charts, Line Graphs, and Data Tables that reflect those metrics instantly.

**[Link to Live Demo](https://vasanthvelumani.github.io/Vasanth-Dashboard-builder/)**

---

## Features

* **Live Workspace Customization**: Features a 12-column grid system where users can drag widgets from a library and drop them onto a canvas to create bespoke layouts.

* **Intelligent Data Visualizations**: Powered by Chart.js, the builder supports multiple visualization types including Area, Scatter, and Pie charts, all of which are configurable via a dedicated settings panel.

* **Real-time Cloud Sync**: Integrated with Firebase Firestore, ensuring that every order submitted and every dashboard configuration saved is persisted across sessions and synced instantly across multiple clients.

* **Granular Widget Control**: Every widget is fully editable. Users can define specific data metrics (Sum, Average, Count), adjust X/Y axes, toggle legends, and even customize UI elements like header colors and font sizes.

* **Secure Environment**: Includes a complete authentication flow (Sign In/Sign Up) powered by Firebase Auth, featuring a "Decrypting Workspace" loader and protected views.

* **Temporal Analytics**: A built-in date filtering system allows users to pivot their entire dashboard view between "Today," "Last 7 Days," or "All Time" with a single click.

## Tech Stack

* **Frontend**: Vanilla JavaScript (ES6+)
* **Database & Backend-as-a-Service**: Firebase Firestore (Real-time data storage and sync)
* **Authentication**: Firebase Auth (Email/Password login system)
* **Data Visualization**: Chart.js (Interactive Bar, Line, Area, and Pie charts)
* **Styling: Custom** CSS3 (Modern UI using CSS Variables, Flexbox, and CSS Grid)
* **Typography**: Inter Font Family via Google Fonts
* **Icons**: Custom SVG icons and CSS-based UI elements
